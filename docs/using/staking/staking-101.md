# Staking 101: PoSV in plain English

PoSV (Proof‑of‑Stake Velocity) is ReddCoin’s way of aligning staking with a **social currency**: it rewards not just holding, but *responsible participation*. This matters for fairness and for avoiding incentives that only benefit the biggest holders.

## Best practices (the Redd Way)
- Stake from a wallet you control (self‑custody).
- Back up your wallet and keys before upgrades or migrations.
- Verify downloads using SHA256.
- If something looks wrong, pause and ask — ReddHeads help each other.
