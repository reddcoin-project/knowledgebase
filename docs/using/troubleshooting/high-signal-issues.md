# High-Signal Recurring Issues

## Exchange Deposit Stuck
1. Verify TXID on official explorer.
2. Confirm correct deposit address.
3. Wait for confirmations.
4. Contact exchange with TXID only.

## Wallet Not Syncing
1. Check peers.
2. Delete peers.dat.
3. Restart or use -reindex.

## No Staking Rewards
- Wallet must be unlocked for staking.
- Coins must be mature.
- Staking is probabilistic.
