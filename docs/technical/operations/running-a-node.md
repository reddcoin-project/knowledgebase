
# Running a ReddCoin Node (Operator Guide)

This guide is for users who want to run a full node and support the network.

## Minimum requirements

- Modern desktop or VPS
- Stable internet connection
- Several GB free disk space (grows over time)
- Router access to open port 45444 (recommended)

## Why run a node?

- Strengthens decentralization
- Improves network propagation
- Helps staking ecosystem stability
- Gives you independent verification of the chain

## Basic steps

1. Install the latest Core wallet
2. Allow full blockchain sync
3. Ensure inbound connections work
4. Keep node online regularly

## Network tips

- Avoid running nodes behind restrictive firewalls
- Ensure system clock is accurate
- Use SSD if possible for faster sync

## Common problems

### Slow sync
- Check disk speed
- Check internet throttling
- Ensure you have multiple peers

### Fork concerns
- Always update to latest wallet
- Verify block height matches explorers
- Restart wallet after updates
