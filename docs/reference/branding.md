# Branding & usage guidelines

This knowledgebase follows the **official Redd / ReddCoin brand guidelines** published at:

- https://redd.love/brand/

If you are creating graphics, listings, articles, or social posts, use the official page above for the latest files and rules.

## Official color palette (from redd.love/brand)

| Name | Hex |
|---|---|
| Dark Red | `#E31B23` |
| Light Red | `#F04D3C` |
| Dark Blue | `#008FC5` |
| Light Blue | `#00AEEF` |
| Dark Orange | `#FDBB30` |
| Light Orange | `#FFD200` |
| Dark Green | `#54B948` |
| Light Green | `#8DC63F` |

## Typography (from redd.love/brand)

- **Goldplay** (logo + major taglines)
- **Rubik** (headings)
- **Roboto** (body text)

## Practical usage notes (for this site)

- The site theme is configured to use **Dark Red** as the primary color and **Light Blue** as the accent.
- Avoid publishing “internal notes” or editor-only markers; this knowledgebase is intended to be public-facing.
