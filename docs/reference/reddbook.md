**Date:** September 2020

PDF: [Redd-Book.pdf](https://redd.love/assets/doc/Redd-Book.pdf)

## What it is

The ReddBook is a project manifesto and organizing document.
It’s less about consensus mechanics and more about **vision, values, and how the community organizes**.
