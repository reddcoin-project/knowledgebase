# PoSV v2 FAQ

PDF: [ReddPaper-PoSV-v2-FAQ.pdf](https://redd.love/assets/doc/PoSV-v2-FAQ.pdf)

## What this FAQ is for

This short document answers the practical questions that come up when a community upgrades to PoSV v2:

- What PoSV v2 *is* in simple terms
- What stakers need to do (wallet upgrade, staking behavior)
- Whether funds can be lost by not upgrading
- What changes for block timing or staking rewards
- Common staking operational questions (pools, expected return)

## Recommended usage

- Link this FAQ from community posts when coordinating upgrades.
- Use the PoSV v2 paper for deeper economic and incentive rationale.
