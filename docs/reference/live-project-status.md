# Live Project Status



## Workstreams

These are suggested groupings to help volunteers and ReddHeads understand what’s active:

- **Core Wallet & Releases** — binaries, upgrades, QA
- **Docs & Knowledgebase** — guides, troubleshooting, onboarding
- **ReddID** — identity layer, namespaces, schema
- **Infra & Explorers** — Blockbook/API endpoints, monitoring
- **Community tooling** — ReddBot, social tipping UX
